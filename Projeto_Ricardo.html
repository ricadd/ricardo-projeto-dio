Introdução

As tags exercem grande importancia em uma documentação HTML. É através delas que conseguimos programar de maneira organizada e detalhada. Por isso, esse projeto irá abordar as principais tags utilizadas em aula, de forma a explicar o funciomento das mesmas, durante a construção de um código.

Como são escritas?

As tags possuem o que chamamos de abertura e fechamento 1 . Isso quer dizer que para criar alguma tag, é necessário abrir e fechar uma tag. Podemos listar algumas das principais tags para a formatação de um texto dentro de um Html. Algumas delas são:

  • Parágrafo: Essas tags são responsáveis por indicar ao computador, onde estará localizado um novo parágrafo. Ou seja, faz com que o computador entenda que determinado trecho trata se de um parágrafo. É indicado pela tag < p >.
  • Break : Essa tag tem uma função de muita utilidade quando estamos digitando um texto e queremos que uma parte dele continue na linha de baixo. Representamos essa tag da seguinte forma: < br >
  • Negrito: Essa tag é bastante utilizada quando queremos enfatizar algum texto. Isso faz com que o leitor perceba melhor uma determinada informação. Há outras tags com o mesmo intuito, mas de início só falaremos dela. Ela é representada pela tag < b > 2 . Exemplo: ''Reunião as 14h sem falta!!''
  • Mark: Como anteriormente falamos sobre deixar o texto em negrito, não poderiamos deixar de falar sobre essa tag. Ela é conhecida por gerar um efeito semelhante ao de um marca texto físico. Essa tag é representada desta forma: < mark > . Exemplo de uso: ''Valor a pagar: R$ 450,00 à vista''
  • Italic : Com referência à famosa Torre de Pisa, essa tag gera um texto levemente inclinado. Usamos a tag < i > para codificá-la.
  • Titulos e subtitulos Para trabalhar com tags que criam textos com tamanhos específicos e com destaque, geralmente utilizados em titulos, utilizamos uma tag que varia do < h1 > ao < h6 > . O que isso quer dizer? Textos com a tag h1 geram textos maiores, enquanto textos com a tag h6 , por exemplo, geram textos minúsculos. Podemos dizer então que o tamanho do texto sempre será inversamente proporcional ao numero utilizado na tag.
  • Small : Essa tag como o próprio nome já diz, deixa o texto pequeno . Para codificá-la, escrevemos da seguinte forma: < small >.
  • Del : Para indicarmos que uma informação não existe mais ou foi alterada, utilizamos uma tag que coloca um risco na palavra. Essa tag se chama del Ex.: O valor atual está na faixa de R$ 300,00 R$ 400,00.

1.: Algumas tags não necessitam de codificarmos um fechamento. Chamamos essas tags de tags com autofechamento.

2.: Se fossemos usar o portugues para escrever as tags, certamente não utilizariamos a letra b para indicar um texto em negrito.
Mas como a base das Tags são todas em ingles, então utilizamos a letra B, que representa bold em ingles. E bold em portugues siginifica negrito.

Outras Tags

Além das principais tags citadas anteriormente, podemos citar outras, tais como sup, sub, strong, font 1 , abbr . A Abbr por exemplo, consegue transformar um determinado titulo em uma forma abreviada. Por exemplo, a palavra Solid State Drive, se transformará em SSD . Com isso, quem está acessando o site, ao passar o cursor por cima da abreviatura, tem como saber o nome completo da palavra. Outra tag também de muita importância é a tag < blockquote > , que tem como finalidade citar um texto de um site externo. O texto citado aparece assim no navegador:

HTML (Hypertext Markup Language) é uma linguagem de marcação utilizada para criar e exibir conteúdo na web. A sua evolução começou em 1989, quando Tim Berners-Lee, um cientista da computação britânico que trabalhava no CERN (Organização Europeia para a Pesquisa Nuclear), desenvolveu a ideia da World Wide Web.

Conclusão

Existe uma infinidade de possibilidade de se trabalhar com as tags em HTML. Além das possibilidades, existe um catálogo enorme de tags disponível na internet. A utilização delas varia bastante pois vai depender do tipo de projeto a ser construido.
